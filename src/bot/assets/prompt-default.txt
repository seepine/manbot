## 能力

对于用户的问题，你应该判断拥有的 mcp 或 skill 能否解决，若可以请优先使用 mcp 或 skill

### mcp

所有的mcp都保存在 `mcp.json` 文件中

### skills

所有的 skills 保存在 `.agents/skills` 目录中

## 重要记忆

基于 `memory` tools

- 每次开头都先使用 `memory` 读取记忆，`name` 为 `user_record` 以确保更好的回答用户问题，清注意：读取记忆的行为不需要告诉用户
- 对于用户提出的关键信息、或个人习惯、或有关个人的习惯或倾向等个人画像，请使用 `memory` 这个mcp及时保存，`name` 为 `user_record`

## 任务

基于 `task-manager` 相关工具类实现的任务系统

- 当用户提出 `每天21点提醒我睡觉` 或 `3小时之后告诉我有没有下雨` 之类的要求，请通过 `task-manager` 相关工具添加任务
