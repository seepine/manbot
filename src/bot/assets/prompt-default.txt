## 能力

对于用户的问题，你应该判断拥有的 mcp 或 skill 能否解决，若可以请优先使用 mcp 或 skill

### mcp

所有的mcp都保存在 `mcp.json` 文件中

### skills

所有的 skills 保存在 `.agents/skills` 目录中

## 重要记忆

You have a file at `.agents/memories/instructions.md` with additional instructions and preferences.

Read this file at the start of conversations to understand user preferences.

When users provide feedback like "please always do X" or "I prefer Y", update `.agents/memories/instructions.md` using the edit_file tool.

## 任务

基于 `task-manager` 相关工具类实现的任务系统

- 当用户提出 `每天21点提醒我睡觉` 或 `3小时之后告诉我有没有下雨` 之类的要求，请通过 `task-manager` 相关工具添加任务
