# 飞书（Lark）集成

## 创建应用

### 1. 创建应用

前往 https://open.feishu.cn/app 创建一个新应用。

### 2. 获取应用 ID 和 Secret

在【基础信息-凭证与基础信息】菜单，获取 `App ID` 和 `App Secret`。

### 3. 添加机器人能力

前往【应用能力-添加应用能力】菜单，添加机器人能力

## 配置应用

### 1. 配置 config.yml

编辑 `config.yml` 文件，在对应 agent 的 `channel` 配置项中设置飞书参数：

| 字段         | 说明                  |
| ------------ | -------------------- |
| `type`       | 通道类型，设为 `feishu` |
| `app-id`     | 飞书应用 App ID       |
| `app-secret` | 飞书应用 App Secret   |
| `app-name`   | 应用名称，用于群聊中判断是否被@ |

详见 [配置文件说明](config.md)。

### 2. 启动程序

启动程序后才能配置事件长链接

### 3. 配置事件订阅方式

前往【开发配置-事件与回调】菜单，在事件配置项中，编辑订阅方式，选择默认的长链接接收事件选项，点击保存

### 4. 添加事件

在下方，添加 接收消息`im.message.receive_v1` 事件，点击保存，并开通以下两个权限

- 读取用户发给机器人的单聊消息
- 接收群聊中@机器人消息事件

### 5. 添加权限

前往 【开发配置-权限管理】菜单，开通以下权限

- 获取与发送单聊、群组消息（im:message）
- 创建与更新卡片（cardkit:card:write）

## 发布应用

前往【应用发布-版本管理与发布】菜单，创建版本并发布应用，此时在飞书中发送消息给机器人，即可进行对话
